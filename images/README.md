# 사진 넣는 곳

이 폴더에 아래 파일명으로 사진을 넣으면 페이지에 바로 반영됩니다.

| 파일명 | 쓰이는 곳 | 추천 비율 |
|---|---|---|
| `og-image.jpg` | 카톡·인스타·구글 공유 미리보기 + 구조화 데이터 대표 이미지 | 가로 1200×630 |
| `yoga-1.jpg` | 갤러리 (세로로 큰 칸) | 세로형 |
| `yoga-2.jpg` | 갤러리 | 정사각/가로 |
| `yoga-3.jpg` | 갤러리 | 정사각/가로 |
| `yoga-4.jpg` | 갤러리 (가로로 넓은 칸) | 가로형 파노라마 |

## 팁
- `og-image.jpg`는 **가장 잘 나온 한 장**으로. 구글 검색·SNS 공유 썸네일에 다 쓰입니다.
- 용량은 장당 **300KB 이하**로 줄이세요 (로딩 속도 = SEO 점수). https://squoosh.app 에서 무료로 압축 가능.
- 사진 더 넣고 싶으면 `index.html`의 `.gallery-grid` 안에 `<figure>` 블록을 복사해서 추가하면 됩니다.
- `alt` 텍스트(이미지 설명)는 영어 키워드를 자연스럽게 — 구글 이미지 검색에 잡힙니다.
